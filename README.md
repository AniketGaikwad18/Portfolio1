# Intelligent Systems Portfolio

A high-end 3D "Scrollytelling" personal portfolio website built with Next.js, React Three Fiber, and Framer Motion.

## 🚀 Features

- **3D Highway Scene**: An immersive driving experience that shifts as you scroll.
- **Machine Learning Showcases**: Dedicated sections for projects, skills, and expertise.
- **Interactive Milestones**: Smooth transitions through Education, Experience, and Certifications.
- **Glassmorphism UI**: Premium visual aesthetic with responsive layouts.
- **Contact Integration**: Live messaging form and social links.

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **3D Library**: Three.js / React Three Fiber
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Icons**: Lucide / SVG

## 🏁 Getting Started

Once you've cloned the repository, install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the live experience.
